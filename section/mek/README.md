# Methyl Ethyl Ketone (MEK) / Butanone ([DIY Practical Effects](../../README.md))

**Methyl Ethyl Ketone** (**MEK**) can be used to perform a _solvent weld_ on certain plastics,
such as [polystyrene](../polystyrene/README.md)
